# Oplo

**Oplo -- Optical Processing, Learning, and Observation**

Computer vision toolbox.

## Quickstart
```bash
python -m venv ~/venvs/oplo && source ~/venvs/oplo/bin/activate
pip install -e .
oplo
```
Open http://127.0.0.1:8050

## Why Oplo
- Supports most scientific image formats
- Interactive (hover probe, histograms)
- Plugin-ready architecture

## Roadmap
- ROI tools (rect/polygon/lasso)
- Timeline player for stacks/videos
- Calibration helpers (checkerboard, rectification)
